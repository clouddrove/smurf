## smurf selm

Subcommand for Helm-related actions

### Synopsis

selm is a subcommand that groups various Helm-related actions under a single command.

```
smurf selm [flags]
```

### Examples

```
smurf selm --help
```

### Options

```
  -h, --help   help for selm
```
