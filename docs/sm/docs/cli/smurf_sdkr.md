## smurf sdkr

Subcommand for Docker-related actions

### Synopsis

sdkr is a subcommand that groups various Docker-related actions under a single command.

```
smurf sdkr [flags]
```

### Examples

```
smurf sdkr --help
```

### Options

```
  -h, --help   help for sdkr
```
