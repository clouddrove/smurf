## smurf selm init

Create a default smurf.yaml file with selm configuration

### Synopsis

This command generates a smurf.yaml file in the current working directory
with default selm configuration placeholders.

```
smurf selm init [flags]
```

### Options

```
  -h, --help   help for init
```
