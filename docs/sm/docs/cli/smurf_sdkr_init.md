## smurf sdkr init

Create a default smurf.yaml file with sdkr configuration

### Synopsis

This command generates a smurf.yaml file in the current working directory
with default sdkr configuration placeholders.

```
smurf sdkr init [flags]
```

### Options

```
  -h, --help   help for init
```
