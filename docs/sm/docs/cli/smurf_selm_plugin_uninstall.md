## smurf selm plugin uninstall

Uninstall one or multiple Helm plugins.

```
smurf selm plugin uninstall [PLUGIN]... [flags]
```

### Examples

```
smurf selm plugin uninstall my-plugin1 my-plugin2
```

### Options

```
  -h, --help   help for uninstall
```
