## smurf stf

Subcommand for Terraform-related actions

### Synopsis

stf is a subcommand that groups various Terraform-related actions under a single command.

```
smurf stf [flags]
```

### Examples

```
smurf stf --help
```

### Options

```
  -h, --help   help for stf
```
